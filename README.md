# Tích hợp Mini-app vào 1 App Android Native

Repo cung cấp tài liệu hướng dẫn về cách tích hợp một Mini-App vào trong một App Android Native

## Cài đặt và chạy

### 1. Clone repository

```bash
git clone https://github.com/HoangHuuTinIT/fake-repo-mini-app-document vb-mini-app-documentation
cd vb-mini-app-documentation
```

### 2. Cài đặt dependencies

```bash
npm install
```

### 3. Chạy development server

```bash
npm run docs:dev
```

Sau khi chạy xong, mở trình duyệt và truy cập: **http://localhost:5173**

### 4. Build production

```bash
npm run docs:build
```

### 5. Preview bản build

```bash
npm run docs:preview
```
